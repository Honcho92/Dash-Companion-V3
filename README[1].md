# Dash Companion v3 — Beta Test

**Store ops tool for 7-Eleven DoorDash orders.**
Built to replace the standard DoorDash tablet app with something faster and built for store employees.

---

## 🔗 Live App
Open on your phone: [https://sevylevy.github.io/dash-companion-v3/](https://sevylevy.github.io/dash-companion-v3/)

---

## 🧪 What to Test

### Orders Tab
- [ ] Tap **Accept** on order #DD-1001 — does the modal pop up correctly?
- [ ] Confirm the accept — does the status update to "Preparing"?
- [ ] Tap **Reject** on an order — does it dismiss cleanly?
- [ ] Tap **⇄ Sub Item** — can you pick a substitute and send it?
- [ ] Tap **Ready** on order #DD-1002 — does the status update?
- [ ] Tap **Picked Up** on order #DD-1003 — does it complete and clear?
- [ ] Watch the order timers — do they count up and turn red after 7 min?

### Inventory Tab
- [ ] Tap any toggle to flip an item in/out of stock
- [ ] Check that the **Out of Stock** counter in the stats bar updates
- [ ] Search for an item (e.g. "monster") — does filtering work?

### Messages Tab
- [ ] Send a message to a customer using the input box
- [ ] Try a **quick reply chip** (e.g. "Order ready!")
- [ ] Send a message to a dasher

### General
- [ ] Does it feel fast and usable on your phone?
- [ ] Is anything confusing or hard to tap?
- [ ] What's missing that you use the DoorDash app for every shift?
